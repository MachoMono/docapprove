# docapprove Wiki

Welcome to the **docapprove** wiki! This is your comprehensive guide to understanding and using the Document Approval Workflow app with Semantic Search.

## What is docapprove?

docapprove is a modern web application built with Next.js and TypeScript that streamlines document approval workflows through intelligent semantic search capabilities. It enables teams to efficiently manage, search, and approve documents with advanced AI-powered features.

- **Documents** - The primary entities in the system that users create and manage
- **Approval Workflows** - Structured processes for document review and approval
- **Semantic Search** - AI-powered search that understands document meaning, not just keywords
- **Users & Permissions** - Role-based access control for document management
